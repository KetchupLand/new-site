Pick your platform

Windows

Download KetchupLand for Windows (x64)

Download KetchupLand for Windows (x32)

KetchupLand is a portable game. You do not need administrator rights, only some file archiving program. Extract and play!

Linux

Download KetchupLand for Linux (AppImage 64-bit) (TODO)

For users of Arch-based distributions, there is an officially maintained AUR package for KetchupLand which should be preferred.

On other distributions you can also choose to clone the source code and follow the instructions to compile the source code.

Android

Download KetchupLand for Android (armeabi-v7a apk)

You will need to sideload it by enabling installation from unknown sources.

Minetest

The fork of Minetest that comes bundled with KetchupLand is the recommended way of playing. But if you already have a regular version of Minetest installed (or are on an unsupported platform) and would like to play KetchupLand through Minetest, you can download and install it through ContentDB or by manually downloading/cloning it from Github.
